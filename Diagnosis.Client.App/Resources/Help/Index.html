Помощь

Diagnosis — это цифровая записная книжка, в которой можно хранить все записи о пациентах, которые вам нужно помнить и использовать.

В этой статье описаны основные действия, которые помогут быстро приступить к работе.

Начало работы

Записи о пациентах располагаются на страницах. С каждым пациентом может быть связано несколько страниц, организованных иерархически — про пациента, курс лечения, осмотр.

Добавление пациентов

  1. Нажмите Все пациенты.

  2. Нажмите кнопку Новый… (CTRL+ВВОД).

    Откроется окно редактирования пациента.

  3. Заполните известные данные.

  4. Нажмите кнопку Далее.

    Для нового пациента создаётся курс и осмотр в нем. Откроется страница первого осмотра.

К содержанию

Добавление курсов

Щелкните правой кнопкой мыши пациента и выберите команду Добавить курс.

Откроется страница нового курса.

При добавлении у курса устанавливается дата начала. Курс можно завершить, указав дату окончания.

Можно добавлять записи на этом уровне или добавить в курс осмотр.
Новые курсы не содержат осмотров.

К содержанию

Добавление осмотров

Щелкните правой кнопкой мыши незавершенный курс и выберите команду Добавить осмотр.

Откроется страница нового осмотра.

При добавлении у осмотра устанавливаются текущие дата и время.

При добавлении пациента сразу добавляется первый осмотр.

К содержанию

Страницы

В левой части экрана отображается иерархия страниц о пациенте — краткие описания пациента, его курсов и осмотров. Текущая открытая страница выделена жирным шрифтом. Последние просмотренные осмотры и курсы подсвечены светлым цветом.

Курсы пациента сортируются от новых к старым, так что первым будет самый последний курс. Порядок сортировки осмотров курса — хронологический, от старых к новым.

На странице есть заголовок и список записей. Вы можете добавлять новые записи, копировать и вставлять записи с этой же или другой страницы, удалять записи, менять представление записей.

Редактирование пациента, курса или осмотра

Щелкните правой кнопкой заголовок или краткое описание страницы и выберите команду Изменить.

Возможно удаление ошибочно добавленных осмотров, курсов и пациентов (без записей на странице и вложенных страниц).

К содержанию

Добавление записей

  1. Откройте страницу, на которой нужно добавить запись.

  2. На панели инструментов нажмите кнопку Новая запись.
    -или-
    Нажмите клавиши CTRL+ВВОД.

    Эта команда также действует в редакторе записей. Вы можете начать ввод новой записи, не закрывая редактор.

В списке появляется выделенная новая запись. Открывается редактор записи с приглашением ввести текст.

К содержанию

Выделение записей

Текущая выбранная запись выделена цветом и отмечена флажком.

Чтобы выделить одну запись, просто щелкните по ней в любом месте.

Двойной щелчок откроет редактор записи.

Чтобы снять выделение, выполните одно из следующих действий.

  • Щелкните пустое место под последней записью в списке.

  • Снимите флажок записи.

  • Щелкните запись, удерживая клавишу CTRL.

Чтобы выбрать несколько последовательных записей, щелкните первую из них, нажмите клавишу SHIFT, а затем щелкните последнюю. Или перемещайте указатель, удерживая нажатой левую кнопку мыши. Появится рамка выделения. Обведите все записи, которые нужно выделить.

Чтобы выбрать несколько непоследовательных записей, нажмите клавишу CTRL и, удерживая ее, щелкните каждую из записей.

Чтобы быстро выделить все записи на странице, нажмите клавиши CTRL+A.

При повторном открытии страницы выделена последняя выбранная запись на этой странице.

К содержанию

Изменение представления записей

Порядок записей на странице определяется сортировкой и группировкой.

Записи сортируются:

  • По порядку

  • По разделу

  • По дате события в записи

  • По дате создания записи

Записи группируются:

  • По разделу

  • По дате создания записи

  • Не группируются

В режиме сортировки по разделу или по дате создания у каждой записи отображается раздел или дата создания (если нет группировки по тому же свойству).

В режиме сортировки по порядку возможно изменение порядка записей вручную.

Если навести указатель мыши на запись, всплывающая подсказка сообщает раздел, даты описания (если указана дата события), создания и изменения, автора записи.

К содержанию

Перемещение записей

Чтобы переместить выделенные записи выше или ниже в списке записей, отсортируйте список записей по порядку и выполните одно из следующих действий.

  • Удерживая нажатой кнопку мыши, перетащите выделенные записи вверх или вниз.

  • Щелкните правой кнопкой мыши по выделенным записям и выберите команду Вверх или Вниз.

  • Нажмите клавиши ALT+SHIFT+СТРЕЛКА ВВЕРХ или ALT+SHIFT+СТРЕЛКА ВНИЗ.

К содержанию

Копирование и вставка записей

  1. Выделите записи, которые нужно скопировать.

  2. Щелкните правой кнопкой мыши по выделенным записям и выберите команду Копировать

  3. Откройте страницу, на которую нужно вставить записи. Щелкните правой кнопкой мыши запись, перед которой нужно добавить записи, и выберите команду Вставить.

    Если на странице нет записей, щелкните правой кнопкой мыши в любом месте страницы.

Скопированные записи добавлены на страницу и выделены.

При копировании дата создания записи меняется на текущую дату.

К содержанию

Удаление записей

Выделите записи, которые нужно удалить, и выполните одно из следующих действий.

  • Нажмите Удалить на панели инструментов.

  • Щелкните правой кнопкой мыши любую из выделенных записей и выберите команду Удалить.

  • Нажмите клавишу DELETE.

Появляется сообщение «Удалено n записей».

Удаленные записи можно восстановить до перехода на другую страницу.

К содержанию

Работа с записью

Запись — строка, описывающая отдельный законченный медицинский факт, наблюдение, предположение, действие. Запись не эквивалентна предложению обычного текста. Запись составляется из элементов некоторого типа. Типы отличаются способом ввода, редактирования, отображением. Для поиска и анализа используются только элементы типа Ключевое слово и Измерение.

Отдельная запись может содержать много элементов, но гораздо эффективнее использовать различные записи. Это упростит поиск записей. Можно указать дату события, которое описывается записью.

Редактирование записи

Чтобы изменить содержимое записи, нужно открыть редактор записи. Для этого выполните одно из следующих действий.

  • Дважды щелкните запись.

  • Выделите запись и нажмите клавишу ВВОД.

Открывается редактор записи с приглашением ввести текст.

Клавишей F2 можно открыть редактор записи без перевода фокуса ввода.

К содержанию

Добавление элементов

Пишите отдельные слова и словосочетания, разделяя их нажатием клавиши ВВОД.

  1. Щелкните пустое место в конце поля ввода и начните писать текст.

    Предлагается список ключевых слов для завершения.

  2. Выберите слово в списке предлагаемых ключевых слов, используя клавиши СТРЕЛКА ВВЕРХ и СТРЕЛКА ВНИЗ.

    Если подходящих ключевых слов нет, введенный текст будет добавлен как Комментарий.

  3. Для подтверждения выбранного слова и завершения ввода нажмите клавишу ВВОД.

    Чтобы вставить текст выбранного слова без завершения ввода, нажмите клавиши CTRL+ПРОБЕЛ.

    Чтобы ввести текст как Комментарий, когда выбрано слово в списке предложений, сначала скройте список, нажав клавишу ESC. Список можно открыть повторно, продолжив ввод текста или нажав клавиши CTRL+ПРОБЕЛ.

  4. Начните ввод следующего слова.

Чтобы добавить текст, для которого не нашлось подходящего словарного слова, как новое ключевое слово, переключите режим завершения ввода слов или используйте клавишу SHIFT при завершении ввода.

Чтобы ввести элемент типа МКБ или измерение, выполните одно из следующих действий.

  • Введите текст в элемент, затем измените тип элемента.

  • Нажмите клавиши CTRL+3 или CTRL+4.

Чтобы добавить элемент среди существующих элементов, выполните одно из следующих действий.

  • Щелкните правой кнопкой мыши элемент и выберите команду Добавить слева или Добавить справа.

  • Переместите добавленный элемент в нужное место после завершения ввода.

К содержанию

Изменение уверенности элемента записи

Уверенность определяет наличие некоторого признака. Уверенность бывает 3 типов:

  • Полная уверенность в наличии признака (по умолчанию для новых элементов).

  • Неуверенность в наличии признака.

  • Полная уверенность в отсутствии признака.

Чтобы изменить уверенность для выделенных элементов, нажмите F8. Уверенность можно менять во время редактирования записи.

В поле ввода отсутствующие элементы надчеркнуты, в списке записей помечены словом «(нет)».

К содержанию

Завершение ввода текста

Для завершения ввода в элемент записи выполните одно из следующих действий.

  • Нажмите клавишу ВВОД.

    Добавляется выбранное ключевое слово или комментарий. Фокус ввода переходит к новому элементу.

  • Нажмите клавиши SHIFT+ВВОД.

    Добавляется комментарий с текстом выбранного ключевого слова или новое ключевое слово. Фокус ввода переходит к новому элементу.

  • Нажмите клавишу F2
    -или-
    Щелкните за пределами поля ввода.

    Если введенный текст совпадает c текстом словарного слова, выбранного в списке предлагаемых слов, добавляется это слово, иначе добавляется комментарий. Редактируемый элемент переходит в состояние выделения.

Редактор элемента записи поддерживает два режима завершения ввода слов: режим завершения и режим предложений. В режиме завершения предлагаются только существующие ключевые слова. В режиме предложений любой напечатанный текст можно добавить как ключевое слово.

Для переключения режима завершения ввода слов выполните одно из следующих действий.

  • Нажмите клавиши CTRL+ALT+ПРОБЕЛ.

  • Нажмите кнопку на панели инструментов редактора записи.

Элементы меняют внешний вид в соответствии с типом:

  • Новые ключевые слова выделяются зеленым цветом.

  • Комментарии — курсивом.

К содержанию

Завершение редактирования записи

Чтобы сохранить изменения в записи, выполните одно из следующих действий.

  • Нажмите клавиши CTRL+S.

    Редактор записи закрывается. Запись остаётся выделенной в списке записей.

  • Снимите выделение записи любым из указанных способов.

    Редактор записи закрывается.

  • Откройте в редакторе другую запись, существующую или новую.

Вы можете добавлять новую запись, не закрывая редактор. Удобнее всего нажать клавиши CTRL+ВВОД.

К содержанию

Выделение и редактирование элементов

Выбранный элемент записи находится в состоянии выделения или редактирования. Выделенные элементы можно перемещать, копировать, удалять, отправлять в поиск.

Нельзя переместить последний (пустой) элемент в поле ввода.

Чтобы выделить элемент, щелкните область слева от текста элемента.

Чтобы редактировать элемент, щелкните область текста элемента. Элементы типа МКБ или Измерение редактируются после двойного щелчка.

Чтобы редактировать уже выделенный элемент, можно нажать клавишу F2.

Когда запись открывается двойным щелчком, редактируется последний (пустой) элемент.

К содержанию

Перемещение элементов

Наведите указатель мыши на область слева от текста элемента. Когда указатель примет вид , переместите элемент в нужное место.

Если переместить элементы в другое поле ввода (например, в поиске), элементы копируются в это поле.
Если переместить элементы на запись в списке записей, элементы добавляются к этой записи.

К содержанию

Изменение типа элемента записи

Элементы записи бывают 4 типов: ключевые слова, комментарии, диагнозы МКБ и измерения. Измерение состоит из ключевого слова и числа с необязательной единицей измерения.

  1. Щелкните правой кнопкой мыши элемент записи.

  2. Выберите нужный тип.

    Если выбран тип МКБ или Измерение, откроется соответствующий редактор.

    При изменении типа с Измерение на Слово числовое значение измерения преобразуется в отдельный элемент-комментарий.

Клавиши CTRL+ALT+<цифра> меняют тип выделенного элемента.

К содержанию

Установка даты записи

Можно указать дату события, которое описывается записью. Поддерживаются точечная дата (момент в прошлом), открытый интервал (стаж), закрытый интервал (длительность). Дата указывается с произвольной точностью — только год, год и месяц, полная дата.

  1. Дважды щелкните запись.

    Откроется редактор записи.

  2. Нажмите Добавить дату и введите дату события любым способом:

    • По известным году, месяцу, дню.

    • Как давность в днях, неделях, месяцах или годах.

    • Как возраст пациента в момент события (доступно, если у пациента указан год рождения).

  3. Выберите вариант отображения даты события в списке записей.

При вводе давности события учитывайте, что фактическая дата рассчитывается относительно даты описания события, которая вводится отдельно. Это нужно, чтобы при вводе «стаж 5 лет» дата события (начало заболевния) не зависела от даты ввода данных.

Чтобы увидеть дату описания записи, наведите указатель мыши на запись, отсортируйте список записей по дате описания или откройте редактор даты.

К содержанию

Ключевые слова

Ключевые слова — термины, которые используются для поиска и анализа записей.

Добавление ключевых слов

  1. Нажмите Словарь.

  2. Введите слово в строке фильтра. Если похожее слово уже есть в словаре, вы увидите его в списке.

  3. Нажмите кнопку Добавить на панели инструментов.
    -или-
    Нажмите клавиши CTRL+ВВОД.

    Откроется редактор слова. Ключевые слова ограничены по длине 100 символами, включая пробелы, и не могут содержать переносы строк.

Ключевые слова можно добавлять и во время редактирования элемента записи.
